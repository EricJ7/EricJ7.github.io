CO2 Monitoring System in TinkerCAD

Designed and Implemented a circuit system in TinkerCAD using multiple Arduino UNOs for a Grade 12 Project.

Circuit consisted of multiple sensors to detect a variety of harmful gases in the atmosphere. Each "slave" Arduino

would send its environmental input to the main Arduino UNO for analysis. If the set threshold was passed for one of

the gases, the main Arduino as well as the "slave" Arduino would begin to buzz at a high frequency and blink their LEDs

Linear Algebra Visualization Tool

Tool for linear algebra to help visualize different vector operations, as well as other applications. Including:

vector addition, subtraction, scalar multiplication, transpose, matrix transformation. etc. Also able to compute the solution to a system

of linear equations with multiple equations in 2d or 3D space. The solutions, as well as operations, are plotted using

matplotlib and Axes3D, Numpy is used to solve the systems and the rest of the operations are implemented from scratch.

Some default transformations are included such as a reflection in the x, or y axis, and a rotation of theta degrees below

the x-axis. The option to view these transformations can be toggled in the GUI

This project is still in progress
